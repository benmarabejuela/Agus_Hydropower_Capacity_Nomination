# Agus_Hydropower_Capacity_Nomination
A repository of my undergraduate thesis codes involving the RTD preprocessing in MATLAB and methodological phases for the LSTM-CQR framework.
AG2_U3_RTD_preprocess is the code I used in MATLAB to preprocess the 5 minutes real-time data of the actual recorded power. For Agus 1, Unit 1 and 2, and Agus 2, Unit 1 and 2, their code is just the same with the Agus 2 Unit 3. You just need to change the name of the file (raw data) that you want to preprocess in the code.
Phase2_CQR and Phase3_CQR are the phases 2 and 3 of Agus 1.
The optimization is also done in one notebook. You will just need to change the parameters and names in the code.
